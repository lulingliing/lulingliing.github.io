css3选择器

子元素选择器(E > F)

  • 子元素选择器 :选中的是子元素F 孙辈不会被选

相邻兄弟选择器(E ＋ F)

  • 只会选择相邻的兄弟级别的
    1.必须是同级的
    2.必须是相邻的
    3.必须是在下方
    4.有且只有一个符合

通用兄弟选择器(E ~ F)

  • 只会选择相邻的兄弟级别的
    1.必须是同级的
    2.必须是在下方
    3.可以有多个符合

通用兄弟选择器：一般用于表单

  • E[attr]
  • E[attr=“value”]
  • E[attr~=“value”] ~：表示几个以空格隔开的属性值中包含了“value”
  • E[attr^=“value”] ^:以value开头的值(ps:可以是一个值也可以是一个类名的首字母)
  • E[attr$=“value”] $:以value结尾的值(ps:是一个类名的最后一个字母或者数字等)

  • E[attr=“value”] *:包含了value的值(ps:由于浏览器的这个星号不识别 所以语句上面有点问题 这边就在这演示一下)

    <style type="text/css">
[class*="div0"]{
color:blue;
border:1px red solid;
}
</style>

<div class="div0">
<h2>我是第一个h2</h2>
<div class="div0div1">
    <h2>我是第二个h2</h2>
    <span>我是第一个span</span>
</div>
<h3>我是第一个h3</h3>
<h3>我是第二个h3</h3>
</div>

  • E[attr|=“value”] |: 以value或value-开头
    E[attr=”value”]和E[attr*=”value”]是最实用的

伪类选择器

锚点伪类
UI元素伪类 表单元素(ps:下面主要介绍的表单元素的伪类)
  • :focus聚焦
  • :disabled表示元素不可用，input元素必须要有disabled属性(ps:表单会暗下去，点击不了)
  • :enabled表示可用状态与disabled完全相反
  • :checked有display：block属性
input:focus{}
input:disabled{}
input:checked{}
input:enabled{
        color: blue;
        }

否定选择器 :not( ) 通常配合属性选择器一起使用

input:not([type="submit"]) {border: 1px solid red;}

意思是除了类型为submit的类型

:nth 选择器

  • :first-child 选择某个元素的第一个子元素
  • :last-child 选择某个元素的最后一个子元素(ps:表示该级别的最后只能是该标签，该标签的同级别下面不能再有别的其他的标签)
  • :nth-last-child()选择某个元素的一个或多个特定的子元素，从这个元素的最后一个子元素开始算；(ps:括号里面填写数字，例：是数字2，则表示倒数第二个)
  • :nth-of-type()选择指定的元素；
  • :nth-last-of-type()选择指定的元素，从元素的最后一个开始计算；（ps:不能跨级对比，在兄弟级别的里面算，不能把父级跟子孙级别的算在一起,表示的是在该级别是最后一个该标签，但是该标签的下面还可以有别的标签）
  • :first-of-type选择一个上级元素下的第一个同类子元素；
  • :last-of-type选择一个上级元素的最后一个同类子元素；
  • :only-child选择的元素是它的父元素的唯一一个子元素；
  • :only-of-type选择一个元素是它的上级元素的唯一一个相同类型的子元素；
  • :empty选择的元素里面没有任何内容。
:nth-child()选择某个元素的一个或多个特定的子元素；
:nth-child(length);/参数是具体数字/(ps:length是具体的数字)
:nth-child(n);/参数是n,n从1开始计算/(ps:里面就是填写n,代表所有的)
:nth-child(2n); /n的倍数选择，n从1开始算/
:nth-child(n+length); /选择大于等于length后面的元素/
:nth-child(-n+length); /选择小于等于length前面的元素/
:nth-child(2n+1); /表示隔几选一/ odd even(ps:odd是奇数 even是偶数)

伪元素

  • ::first-letter 选择文本块的第一个字母，除非在同一行里面包含一些其它元素，不过这个主要运用于段落排版上多

  • ::first-line 选择元素的第一行

  • ::before,::after 这两个主要用来给元素的前面或后面插入内容，这两个常用”content”配合使用，见过最多的就是清除浮动还有图片icon

  • ::selection用来改变浏览网页选中文本的默认效果“::selection”来改变在浏览器中选中文本后的背景色与前景色。::selection在IE家族中，只有IE9+版本支持，在Firefox中需要加上其前缀“-moz”，:“::selection”只能设置两个属性，一个就是background,另一个就是color属性，设置其他属性是没有任何效果的。

css3盒子模型

box-sizing:content-box(默认值) || border-box（包含边框和padding） || padding-box(ps:padding-box只有在火狐浏览器才支持，而且也只支持pc端，border-box在IE浏览器不支持)
    <style type="text/css">
    div{
        background: red;
        font-size: 45px;
        color: white;
        width: 300px;
        height:300px;
        }
</style>

    <div style="box-sizing:border-box;outline:10px         blue solid;border:10px pink solid;">
        我是div1
    </div>
    <hr>
    <div>我是div2</div>

  • 设置了border-box，就算设置了边框也是包含在内容里面的，不会改变原来盒子的宽高
  • 设置了padding-box之后，设置padding也不会影响容器的大小。

短语句

  • mark(标记):高亮文本，常作用于搜索高亮

  • meter(定义度量衡):表示在一定范围内的值，如投票占比，使用量等。

    属性: value:实际度量的值，默认为0，可以为整数或者浮点数。 high:范围的上限值 low:范围的下限值 max:最大值，默认值是1 min:最小值，默认值是0 optimum:最佳的值，必须在min属性值与max属性值之间

  • progress(定义进度):属性 value max min

  • time(定义时间和日期):
<p>We open at <time>10:00</time> every morning.</p>

<p>I have a date on <time datetime=“2008-02-14”>Valentines day</time>.</p>

datetime=“YYYY-MM-DDThh:mm:ssTZD”

  • abbr(定义缩写):

    The <abbr title=“World Health Organization”>WHO</abbr> was founded in 1948.

  • bdi(定义隔离)

  • cite:引入(字是斜体)
  • blockquote:引入(长文本引入)
  • q:引入(短文本引入),内容会由引号括起来
  • output:输出
  • code:格式化代码
  • del:删除
  • ins:下划线

表单元素

input

1.tel:用于输入电话号码。
2.search:用于提示用户输入要搜索的文字。search 和 text 之间的区别主要在于样式上
3.url:用于输入单个网址。
4.email:用于输入单个电子邮件地址。如果指定了 multiple 属性，就可以输入多个电子邮件地址，以逗号分隔
5.datetime:用于输入时区设置为 UTC 的日期和时间。
6.date:用于输入不含时区的日期
7.month:用于输入含年份和月份但不含时区的日期。
8.week:用于输入含年份和周数但不含时区的日期。例如，此格式的日期 2011-W05 表示 2011 年的第 5 周。
9.time:用于输入含小时、分钟、秒和秒的小数部分但不含时区的时间值。
10.datetime-local:用于输入不含时区的日期和时间。
11.number:用户输入数字
12.range:用于输入数字，但与 number 的区别在于无需输入具体数字。在大部分支持该类型的浏览器中，范围控件的实施形式为滑块。
13.color:用于通过颜色池控件选择颜色。

label:主要用于登录密码的时候

datalist:配合input使用,list属性配合id名

fieldset legend:fieldset 是用一个大框把input包含起来 legend是边框的标题(字在框上)

placeholder:用于提示用户应输入的数据类型。 在对焦到相关元素以及用户输入数据之前，系统会以浅色文字显示占位符值。您可以在输入和文本区域中指定该值。
autocomplete:纪律搜索记录，可用于搜索框
required:监测，例：邮箱的话里面是必须有@的，不写的时候就会出现提示，不符合要求

国内双核浏览器默认内核模式如下：

  • 搜狗高速浏览器、QQ浏览器：IE内核（兼容模式）
  • 360极速浏览器、遨游浏览器：Webkit内核（极速模式）

搜索引擎索引方式

<meta name=“robots” content="index,follow" />
  • all：文件将被检索，且页面上的链接可以被查询；
  • none：文件将不被检索，且页面上的链接不可以被查询；
  • index：文件将被检索；
  • follow：页面上的链接可以被查询；
  • noindex：文件将不被检索；
  • nofollow：页面上的链接不可以被查询。

meta标签

meta提供关于HTML文档的元数据。原数据不会显示在页面上，但是对于机器是可读的。它可用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 web 服务。

1.声明编码

<meta charset='utf-8' />

2.优先使用 IE 最新版本和 Chrome

<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1" />

3.浏览器内核控制

国内浏览器很多都是双内核（webkit和Trident），webkit内核高速浏览，IE内核兼容网页和旧版网站。而添加meta标签的网站可以控制浏览器选择何种内核渲染。 

<meta name="renderer" content="webkit|ie-comp|ie-stand">

4.页面关键字

<meta name="keywords" content="your tags，(例:袜子，电脑，电子产品)” />

5.页面描述

<meta name=“description"content="120｜ 150 words(包括标点符号)" />

6.页面重定向和刷新

<meta http-equiv="refresh" content="0;url=" />

content内的数字代表时间（秒），既多少时间后刷新，如果加url,则会重定向到指定网页（搜索引擎能够自动检测，也很容易被引擎视作误导而受到惩罚）

7.网页作者信息

<meta name="author" content="author name" />

8.移动设备

<meta name="viewport" content="width=device-width, initial-scale=1.0,maximum-scale=1.0, user-scalable=no"/>

9.WebApp全屏模式：伪装app，离线应用。

<meta name="apple-mobile-web-app-capable" content="yes" />

10.隐藏状态栏/设置状态栏颜色：只有在开启WebApp全屏模式时才生效

<meta name="apple-mobile-web-app-status-bar-style"content="black-translucent" />

11.添加到主屏后的标题

<meta name="apple-mobile-web-app-title"content="标题">

content的值为default | black | black-translucent

12.忽略数字自动识别为电话号码

<meta content="telephone=no"name="format-detection" />

13.忽略识别邮箱

<meta content="email=no"name="format-detection" />

css3的单位与响应式布局

倍数单位

1. em是相对单位

  • 相对父级设置和自己的字号设置

注意：h1~h6都有默认的1.5em,当不给h标签设置font-size，就会按照父级的1.5em计算，
当给h2设置了font-size,就会按照设置的倍数计算

<pre>
<style>
    .nav{
        width:500px;
        height:300px;
        border:1px red solid;
        font-size:12px;
    }
    .nav p{
        font-size:4em;
    }
    .nav h2{
        font-size:2em;
    }
</style>
<div class=“nav”>
    <p>世界你好</p>
    <h2>一切皆有可能</h2>
</div>
</pre>

上面的这样计算下来p标签的字号大小为：124=48px;h2标签的字号大小为122=24px，但是如果不给h2设置fong-size就会按照父级的1.5em设置，也就是12*1.5=18px;

2.rem是根单位

  • 相对于根字体(ps:根据HTML的字号）的大小来设置自己的字号大小

注意：系统默认的为16px(当HTML没有设置字体的大小就会按照默认的16px计算)，系统最小的字号为12px(ps:当在HTML中设置的字体小于12px,系统会按照最小的12px计算)

百分比单位

  • vw,vh是相对于窗口的单位(指宽度跟高度)

  • maxwidth maxheight minwidth minheight 这些是设置的最大的跟最小的宽高度（ps:不是百分比单位，是设置具体的像素）

搭建hexo博客

准备工作

nodeJS

首先下载安装一个nodeJS，也可以通过nodeJS下载地址，测试nodeJS是否安装成功，在终端命令行输入node -v，出现v+数字，例：v4.4.5就表示安装成功啦！

git

下载安装一个git,在终端输入git –version,出现git version+数字，例：git version 2.7.4，就表示已经安装好了，接下来就是搭建hexo了。

步骤：

安装本地hexo

1.在终端输入npm install hexo -g 回车，如果报错出现root/Administrator,则输入node npm install hexo -g 回车，则出现如下图，输入管理员密码，回车

2.测试hexo是否安装成功，在终端输入hexo回车,如若出现如下图所示的代码，则安装成功。

新建一个文件夹（ps:以桌面创建为例）

1.在终端输入cd Desktop回车，进入电脑桌面

2.在终端输入mkdir blog回车，创建了一个名为blog的文件夹

3.在终端输入cd blog回车，进入该文件夹

4.当前的blog是空的，在终端输入hexo init,回车，就会出现很多文件夹，初始化blog文件夹，陈成功之后就会是如下图：

5.在终端输入hexo server回车，在本地运行，成功之后在终端会出现一个本地博客地址http://localhost:4000/，点击进入。按住Ctrl+c退出hexo运行

6.在终端输入hexo g回车，生成HTML文件

7.在终端输入hexo d回车，发布到线上

注册一个账号

1.在注册界面注册一个用户名，填写一个邮箱与密码，然后直接进入里面

2.注册一个github账号，在终端输入ssh-keygen,连续回车三次，如下图：

3.在终端输入vim ~/.ssh/id_rsa.pub命令，复制ssh-key的值（ps:一个波浪线都不要多复制），在终端输入：q回车，打开本地博客，点击settings,进入里面的SSH and GPG keys，在key的框里将ssh-key的值粘贴进去。

验证本机机器与github

1.在终端输入“git config –global user.name”用户名”（例：git config –global user.name”lulingliing”）

2.在终端输入“git config –global user.email”邮箱地址”（例：git config –global user.email”1552442072@qq.com”）

3.在本地博客搭建域名仓库，repository必须与注册的用户名一致。

配置config.yml

1.在blog文件夹中的config.yml文件下的deploy下写空格+git

2.在本地博客中将域名仓库HTTPS的地址写在repo下（ps:在config.yml中写代码都需要在冒号后面加空格）如下图：
