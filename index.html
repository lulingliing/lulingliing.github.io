怪异模式盒模型（quirks 模式）

  • 含义：在怪异模式下的盒模型，盒子的总宽度和高度是包含内边距padding和边框border宽度在内的，
  • 一个块的总宽度= width + margin(左右)（即width已经包含了padding和border值）

在标准模式下的盒模型:

  • 一个块的总宽度= width + margin(左右) + padding(左右) + border(左右)

  • 也就是设置padding的时候就增加盒子的宽高大小

box-sizing

  • -webkit-box-sizing有两个值一个是content-box，另一个是border-box

    • 当设置为box-sizing:content-box时，将采用标准模式解析计算， 也是默认模式；
    • 当设置为box-sizing:border-box时，将采用怪异模式解析计算；

    }

resize:改变元素尺寸大小（ps:IE/欧朋不支持）

  • resize:none | both | horizontal | vertical | inherit

    • none:不能修改

    • both:可以同时修改宽高

    • horizontal:只能修改宽度

    • vertical:只能更改高度

  • 语法：

<style>

        resize:none;
        }
    div{
        width: 200px;
        height: 200px;
        background: #ccc;
        resize: both;
        overflow: auto;/*自动有滚动条*/
        }
</style>

<body>

    <div></div>
</body>
注意：如果给div一个元素一开始设置了宽高，想改变它的宽高就必须加overflow：hidden，就想上面的div一样。

transition 过渡

1.transition-property:

  • 执行过渡变换的属性（all 、width、height、left、right、opacity、边框的宽度支持，solid不支持【类型不支持】、font-size、color。。。）

2.transition-duration：

  • 过渡变换延续的时间（单位是s，可以为小数）

3.transition-timing-function

  • 过渡的变化类型（过渡变换的速率效果方式）

  • transition-timing-function类型

    • (1) ease：先快后慢

    • (2) ease-in：先慢后快

    • (3) ease-in-out:慢—>快—>慢

    • (4) ease-out：先快后慢

    • (5) linear（匀速运动）

    • (6)用贝塞尔曲线来定义

4.transition-delay:

  • 过渡变换延迟时间
简写方式
transition:all .5s ease-in-out 1s;
含义：所有属性在1秒后以慢—>快—>慢的方式播放，播放一次的时间为0.5秒

[注意：设置过渡的时候最好设置一个初始值,transition的效果是需要用 hover才能实现]

transform2D 变形

transform: rotate 旋转（deg角度）

  • 通过指定的角度参数对原元素指定一个2D rotation（2D 旋转）

  • deg-可为负数

  • 可以有X\Y\Z轴(默认为Z轴)

transform: scale 缩放（倍数）

  • 倍数可以为整数 小数（可以为负数，出现的是颠倒过来的效果）,水平方向跟垂直方向都是同时缩放

    • scale(2,0.5)：元素水平方向放大到原来2倍，垂直方向缩小到原来的0.5倍

    • scale(2)：元素水平方向跟垂直方向同时放大到原来的2倍

    • scaleX(0.3)：元素仅水平方向缩小到原来的0.3倍

    • scaleY(2)：元素仅垂直方向放大到原来的2倍

transform: skew 扭曲（deg角度）

  • 角度可以设置为正数，也可以设置为负数,水平方向跟垂直方向都是同时扭曲

    • skew(30deg,45deg):元素在水平扭曲30deg,垂直方向扭曲45deg

    • skew(30deg):水平垂直方向同时扭曲30deg

    • skewX(30deg):仅使元素在水平方向扭曲30deg

    • skewY(45deg):仅使元素在垂直方向扭曲变形45deg

transform:translate 移动（像素或者百分比）

  • 像素可以为负数，百分比是占本身的比例，translate()，必须设置两个值，不能用一个值来简写代表两个，水平方向跟垂直方向都是同时移动

    • translate(50px,100px):先同时(x轴向右，y轴向下,因为y轴的正方向在下面)向右下角移动50px,然后y轴方向再向下移动50px

    • translate(50%,50%):x轴，y轴同时向右下角移动盒子本身的50%的距离

    • translateX(x)仅水平（x轴）方向移动

    • translateY(Y)仅垂直（y轴）方向移动

transform-origin:X Y

  • 旋转元素的基点（旋转点），默认的旋转点在元素的中心位置

    • 其中X和Y的值可以是百分比,em,px，方向

    • transform-origin:20px(当设置为一个值的时候，表示的是沿x轴平移20px，y轴默认为中心位置)

    • transform-origin:left(当设置的为方向的时候，表示沿着某一条边旋转)

transform简写方式；
transform: rotate() scale() skew() translate(),可以调换顺序
（旋转rotate、缩放scale、扭曲skew、移动translate）

transform：rotate（旋转）

  • rotate(45deg)：在不给确定的方向值时，默认绕z轴旋转

  • rotateX(45deg)：元素围绕其 X 轴旋转45度（从X轴正方向【从右边】看是顺时针旋转）

  • rotateY(30deg)：元素围绕其 Y 轴旋转30度(从y轴正方向【从下往上，因为y轴正方向在下面】看)

  • rotateY(30deg)：元素围绕其 Z 轴旋转30度（顺时针旋转）

【注意：旋转属性可以设置多个方向的，但是translate与rotate一起使用的时候，碧玺把translate写在前面，都写在同一行，z轴跟x轴或者y轴一起使用的时候，不用写z轴，按照默认的，例：transform：translateZ（200px） rotate(15deg) rotateY(90deg)】

transform-style:

  • 含义：嵌套元素如何在3D空间中呈现

    • transform-style: flat | preserve-3d

    • flat默认，所有子元素在2D平面呈现

    • preserve-3d，所有子元素在3D平面呈现，【注意：设置这个属性之后，就不能overflow:hidden】

perspective:视距，视角 设置查看者的位置（ps:当做舞台，视距可以设置的大一点）

  • 两种书写方式：

    • 可以单独直接设置：perspective：400px

    • 也可以在transform里面设置：transform:perspective|(400px),rotateY(45deg);

【注意：一旦在z轴设置移动的值超过了视距，就会看不见该元素了】

perspective-origin 视角中心(观看点)

  • perspective-origin: x y;可以设置为像素跟百分比，只能设置x轴跟y轴的

transform-origin:

  • 含义：旋转元素的基点（也就是该元素是围绕该点旋转的）

  • 3D可以设置三个属性值，分别是x轴，y轴，z轴

【注意;x轴y轴都可以设置为像素，百分比和em，但是z轴不能设置为百分比，只能设置像素跟em】

transform:translateZ(100px);

  • 沿z轴移动，只能设置像素值，可正可负，（由于z轴正方向是正对着我们，设置正数移动的时候有放大的效果，设置负数移动有缩小的效果）

backface-visibility(元素旋转到背面是否可见)

  • backface-visibility: visible（可见） | hidden（隐藏）

animation 动画效果

1.animation-name /动画属性名/

2.@keyframes 动画关键帧

  • 用from和to来写；例：

    @keyframes name{
    from{
        width: 200px;
        }
    to{
        width: 100px;
        }
    }
  • 用(百分比)10%，20%，30%。。。100%，来写，例：
@keyframes name{
    0%{
        transform: scale(1);
    }
    50%{
        transform:scale(1.5,1.5);
    }
    100%{
        transform:scale(1);
    }
}

3.animation-duration /动画持续时间/，单位：s

4.animation-timing-function /动画效果/

  • 动画类型：ease | linear | ease-in | ease-out | ease-in-out | cubic-bezier

5.animation-iteration-count /指定元素播放动画的循环次数/

  • infinite 无限；数字代表几次

6.animation-delay 指定元素动画开始时间，单位： s秒

7.animation-direction /指定元素动画播放的方向/

  • 默认值为normal，动画的每次循环都是向前播放

  • alternate，动画播放在第偶数次向前播放，第奇数次向反方向播放

8.animation-fill-mode：forwrds

  • 含义：保留动画的最后一个属性值， 会在最后一个属性值停住
动画效果简写方式
animation: haha 5s infinite linear 1s;
（动画名称，持续时间，播放次数，动画速度变化方式，动画开始的延迟时间）
含义：动画名为haha在1秒之后无限循环的播放，播放持续时间为5s
【注意：transition与animation不同的是，transition是需要hover实现，而animation是自动效果】

css3的圆角设置和背景设置

css3盒子机制（box-sizing）

  • box-sizing:border-box(怪异模式)

    • border和padding的值都是计算在width内的

  • box-sizing:content-box(标准盒子模型，在不设置盒子模型的情况下默认该种盒子模型)

    • border和padding不计算入width内

  • box-sizing:padding-box

    • padding计算在width内，border不计算在内

border设置

圆角设置

  • border-radius:10px 20px 30px 40px/50px 60px 70px 80px;

  • 含义：左上水平半径 右上水平半径 右下水平半径 左下水平半径/左上垂直半径 右上垂直半径 右下垂直半径 左下垂直半径

  • 值可以设置为px跟百分比，设置半分比的时候就是相对于盒子的百分比

阴影设置

  • box-shadow:3px 4px 5px #000 inset;(text-shadow: 4px 4px 5px black;)
  • 含义：x-offset偏移3像素的阴影 y-offset偏移4像素的阴影 模糊度为5像素 阴影颜色设置为黑色 inset为内阴影
  • box-shadow为盒子的阴影，text-shadow为文字阴影，不同的是文字不能设置内阴影，也就是没有inset属性

背景设置

background-origin：content-box || padding-box || border-box

  • content-box

    • 设置了padding之后，图片不会盖住padding，现在就是相当于背景图跟内容一样，padding了之后是对背景图是有影响的（box-sizing:content-box表示width不包含border和padding，也就是我们以前设置的盒子模型）

    • padding-box

      • 默认的就是padding-box的形式，也就是背景图是盖住padding的，换句话说看上去就是设置padding对背景图片是没有变化的（box-sizing:padding-box表示width不包含border，包含padding）

    • border-box

      • 边框会把背景图给盖住，因为图片是从左上角最顶端开始的（box-sizing:border-box表示width既包含border也包含padding）
background-clip：content-box || padding-box || border-box
  • clip中之后content-box有效果，配合padding使用，以内容范围裁剪（剩下的两个值设置的效果是标准盒子下的背景图片设置机制，相当于 background-origin：padding-box ）

渐变

background-img:linear-gradient(线性渐变)

  • repeating-linear-gradient (to left,red 20%,blue 80%)

    • 含义：重复的线性渐变,（从右）向左，由红色向蓝色渐变，ps:红色在右边

  • linear-gradient（to top left,red 20%,blue 80% ）

    • 含义：（从右下角）到左上角，由红色向蓝色渐变，ps:红色在右下角

  • linear-gradient（45deg, red 20%,blue 80% ）

    • 注意：线性渐变设置角度的时候是顺时针旋转

  • 属性值解析：渐变的方向，初始颜色值，终止颜色值（颜色值的百分比可以设置为像素，注意终止颜色值的像素值或者是百分比必须大于初始颜色值）

  • 可以运用background-size于repeat做出网格效果；例：

    background-image: linear-gradient(to right,pink                 50%,transparent 50%),linear-gradient(to             bottom,blue 50%,black 50%);
background-size:50px 50px; 
background-repeat:repeat-x;

注意：第一个必须使用transparent（透明色），因为后面的颜色会把前面的颜色盖住

background-img:radial-gradient(径向渐变)【at是发散范围 at后面是发散的中心点 起止颜色跟终止颜色】

  • background-image:radial-gradient(circle（ellipse）,red 20%,blue 80%);

    • 含义：以圆形（椭圆）形式从中心，红色实体占20%开始向蓝色渐变
注意：渐变的形状，ellipse表示椭圆形，circle表示圆形。默认为ellipse，如果元素形状为正方形的元素，则ellipse和circle显示一样

  • background-image:radial-gradient(at center right,red 20%,blue 80%);

    • 含义:at center right指的是从中间最后边的位置，从红色渐变到蓝色，红色实体色占百分之二十

  • background-image:radial-gradient( 50px 50px at 10px 10px ,red 20% ,blue 80%);

    • 含义：渐变后的形状宽高度为50px,在容器的10px 10px的位置处为中心点，从红色渐变到蓝色，红色实体色为20%

css3选择器

子元素选择器(E > F)

  • 子元素选择器 :选中的是子元素F 孙辈不会被选

相邻兄弟选择器(E ＋ F)

  • 只会选择相邻的兄弟级别的
    1.必须是同级的
    2.必须是相邻的
    3.必须是在下方
    4.有且只有一个符合

通用兄弟选择器(E ~ F)

  • 只会选择相邻的兄弟级别的
    1.必须是同级的
    2.必须是在下方
    3.可以有多个符合

通用兄弟选择器：一般用于表单

  • E[attr]

  • E[attr=“value”]

  • E[attr~=“value”] ~：表示几个以空格隔开的属性值中包含了“value”

  • E[attr^=“value”] ^:以value开头的值(ps:可以是一个值也可以是一个类名的首字母)

  • E[attr$=“value”] $:以value结尾的值(ps:是一个类名的最后一个字母或者数字等)

  • E[attr=“value”] *:包含了value的值(ps:由于浏览器的这个星号不识别 所以语句上面有点问题 这边就在这演示一下)

    <style type="text/css">
[class*="div0"]{
color:blue;
border:1px red solid;
}
</style>

    <h2>我是第二个h2</h2>
    <span>我是第一个span</span>
</div>
<h3>我是第一个h3</h3>
<h3>我是第二个h3</h3>
</div>

  • E[attr|=“value”] |: 以value或value-开头
    E[attr=”value”]和E[attr*=”value”]是最实用的

伪类选择器

锚点伪类
UI元素伪类 表单元素(ps:下面主要介绍的表单元素的伪类)

  • :focus聚焦

  • :disabled表示元素不可用，input元素必须要有disabled属性(ps:表单会暗下去，点击不了)

  • :enabled表示可用状态与disabled完全相反

  • :checked有display：block属性

        }

否定选择器 :not( ) 通常配合属性选择器一起使用

input:not([type="submit"]) {border: 1px solid red;}

意思是除了类型为submit的类型

:nth 选择器

  • :first-child 选择某个元素的第一个子元素

  • :last-child 选择某个元素的最后一个子元素(ps:表示该级别的最后只能是该标签，该标签的同级别下面不能再有别的其他的标签)

  • :nth-last-child()选择某个元素的一个或多个特定的子元素，从这个元素的最后一个子元素开始算；(ps:括号里面填写数字，例：是数字2，则表示倒数第二个)

  • :nth-of-type()选择指定的元素；

  • :nth-last-of-type()选择指定的元素，从元素的最后一个开始计算；（ps:不能跨级对比，在兄弟级别的里面算，不能把父级跟子孙级别的算在一起,表示的是在该级别是最后一个该标签，但是该标签的下面还可以有别的标签）

  • :first-of-type选择一个上级元素下的第一个同类子元素；

  • :last-of-type选择一个上级元素的最后一个同类子元素；

  • :only-child选择的元素是它的父元素的唯一一个子元素；

  • :only-of-type选择一个元素是它的上级元素的唯一一个相同类型的子元素；

  • :empty选择的元素里面没有任何内容。

:nth-child()选择某个元素的一个或多个特定的子元素；
:nth-child(length);/参数是具体数字/(ps:length是具体的数字)
:nth-child(n);/参数是n,n从1开始计算/(ps:里面就是填写n,代表所有的)
:nth-child(2n); /n的倍数选择，n从1开始算/
:nth-child(n+length); /选择大于等于length后面的元素/
:nth-child(-n+length); /选择小于等于length前面的元素/
:nth-child(2n+1); /表示隔几选一/ odd even(ps:odd是奇数 even是偶数)

伪元素

  • ::first-letter 选择文本块的第一个字母，除非在同一行里面包含一些其它元素，不过这个主要运用于段落排版上多

  • ::first-line 选择元素的第一行

  • ::before,::after 这两个主要用来给元素的前面或后面插入内容，这两个常用”content”配合使用，见过最多的就是清除浮动还有图片icon

  • ::selection用来改变浏览网页选中文本的默认效果“::selection”来改变在浏览器中选中文本后的背景色与前景色。::selection在IE家族中，只有IE9+版本支持，在Firefox中需要加上其前缀“-moz”，:“::selection”只能设置两个属性，一个就是background,另一个就是color属性，设置其他属性是没有任何效果的。

css3盒子模型

box-sizing:content-box(默认值) || border-box（包含边框和padding） || padding-box(ps:padding-box只有在火狐浏览器才支持，而且也只支持pc端，border-box在IE浏览器不支持)
    <style type="text/css">

    <div style="box-sizing:border-box;outline:10px         blue solid;border:10px pink solid;">
        我是div1
    </div>
    <hr>
    <div>我是div2</div>

  • 设置了border-box，就算设置了边框也是包含在内容里面的，不会改变原来盒子的宽高

  • 设置了padding-box之后，设置padding也不会影响容器的大小。

  • 360极速浏览器、遨游浏览器：Webkit内核（极速模式）

搜索引擎索引方式

<meta name=“robots” content="index,follow" />
  • all：文件将被检索，且页面上的链接可以被查询；
  • none：文件将不被检索，且页面上的链接不可以被查询；
  • index：文件将被检索；
  • follow：页面上的链接可以被查询；
  • noindex：文件将不被检索；
  • nofollow：页面上的链接不可以被查询。

meta标签

meta提供关于HTML文档的元数据。原数据不会显示在页面上，但是对于机器是可读的。它可用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 web 服务。

1.声明编码

<meta charset='utf-8' />

2.优先使用 IE 最新版本和 Chrome

<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1" />

3.浏览器内核控制

国内浏览器很多都是双内核（webkit和Trident），webkit内核高速浏览，IE内核兼容网页和旧版网站。而添加meta标签的网站可以控制浏览器选择何种内核渲染。 

<meta name="renderer" content="webkit|ie-comp|ie-stand">

4.页面关键字

<meta name="keywords" content="your tags，(例:袜子，电脑，电子产品)” />

5.页面描述

<meta name=“description"content="120｜ 150 words(包括标点符号)" />

6.页面重定向和刷新

<meta http-equiv="refresh" content="0;url=" />

content内的数字代表时间（秒），既多少时间后刷新，如果加url,则会重定向到指定网页（搜索引擎能够自动检测，也很容易被引擎视作误导而受到惩罚）

7.网页作者信息

<meta name="author" content="author name" />

8.移动设备

<meta name="viewport" content="width=device-width, initial-scale=1.0,maximum-scale=1.0, user-scalable=no"/>

9.WebApp全屏模式：伪装app，离线应用。

<meta name="apple-mobile-web-app-capable" content="yes" />

10.隐藏状态栏/设置状态栏颜色：只有在开启WebApp全屏模式时才生效

<meta name="apple-mobile-web-app-status-bar-style"content="black-translucent" />

11.添加到主屏后的标题

<meta name="apple-mobile-web-app-title"content="标题">

content的值为default | black | black-translucent

12.忽略数字自动识别为电话号码

<meta content="telephone=no"name="format-detection" />

13.忽略识别邮箱

<meta content="email=no"name="format-detection" />

css3的单位与响应式布局

倍数单位

1. em是相对单位

  • 相对父级设置和自己的字号设置

注意：h1~h6都有默认的1.5em,当不给h标签设置font-size，就会按照父级的1.5em计算，
当给h2设置了font-size,就会按照设置的倍数计算

<pre>
<style>
    .nav{
        width:500px;
        height:300px;
        border:1px red solid;
        font-size:12px;
    }
    .nav p{
        font-size:4em;
    }
    .nav h2{
        font-size:2em;
    }
</style>
<div class=“nav”>
    <p>世界你好</p>
    <h2>一切皆有可能</h2>
</div>
</pre>

上面的这样计算下来p标签的字号大小为：124=48px;h2标签的字号大小为122=24px，但是如果不给h2设置fong-size就会按照父级的1.5em设置，也就是12*1.5=18px;

2.rem是根单位

  • 相对于根字体(ps:根据HTML的字号）的大小来设置自己的字号大小

注意：系统默认的为16px(当HTML没有设置字体的大小就会按照默认的16px计算)，系统最小的字号为12px(ps:当在HTML中设置的字体小于12px,系统会按照最小的12px计算)

百分比单位

  • vw,vh是相对于窗口的单位(指宽度跟高度)

  • maxwidth maxheight minwidth minheight 这些是设置的最大的跟最小的宽高度（ps:不是百分比单位，是设置具体的像素）

nodeJS

首先下载安装一个nodeJS，也可以通过nodeJS下载地址，测试nodeJS是否安装成功，在终端命令行输入node -v，出现v+数字，例：v4.4.5就表示安装成功啦！

git

下载安装一个git,在终端输入git –version,出现git version+数字，例：git version 2.7.4，就表示已经安装好了，接下来就是搭建hexo了。
