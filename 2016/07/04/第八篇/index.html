怪异模式盒模型（quirks 模式）

  • 含义：在怪异模式下的盒模型，盒子的总宽度和高度是包含内边距padding和边框border宽度在内的，
  • 一个块的总宽度= width + margin(左右)（即width已经包含了padding和border值）

在标准模式下的盒模型:

  • 一个块的总宽度= width + margin(左右) + padding(左右) + border(左右)

  • 也就是设置padding的时候就增加盒子的宽高大小

box-sizing

  • -webkit-box-sizing有两个值一个是content-box，另一个是border-box

    • 当设置为box-sizing:content-box时，将采用标准模式解析计算， 也是默认模式；
    • 当设置为box-sizing:border-box时，将采用怪异模式解析计算；

css3 @font-face 规则

引入字体

语法：
    @font-face ：{
    font-family:<你的web字体名称>;
    src:url(“字体路径”);
    font-weight:bold;
}
引用：
div{
    font-family:你的web字体名称;
    }

resize:改变元素尺寸大小（ps:IE/欧朋不支持）

  • resize:none | both | horizontal | vertical | inherit

    • none:不能修改

    • both:可以同时修改宽高

    • horizontal:只能修改宽度

    • vertical:只能更改高度

  • 语法：

<style>
    textarea{
        resize:none;
        }
    div{
        width: 200px;
        height: 200px;
        background: #ccc;
        resize: both;
        overflow: auto;/*自动有滚动条*/
        }
</style>

<body>
    <textarea></textarea>
    <div></div>
</body>
注意：如果给div一个元素一开始设置了宽高，想改变它的宽高就必须加overflow：hidden，就想上面的div一样。

column 多列（ps:只能用于文字，不能用于英文，中文加英文都不行）

  • column-count：定义分栏的数目。
  • column-gap：定义列之间的间隔。

  • column-rule：定义列与列之间间隔线的宽度，样式和颜色。

    div{
-webkit-column-count:3; 
-webkit-column-gap:40px; 
-webkit-column-rule:3px red solid; 
}

css3 弹性布局

box-flex :让子容器针对父容器的宽度按一定规则进行划分（注意：使用时父级必须设置display:box；）

语法：
    <style>
        .wrap{
        display:-webkit-box;
            }
        .wrap div:nth-child(1){
            background: #ccc;
            -webkit-box-flex:3;
            }
        .wrap div:nth-child(2){
            background: green;
            -webkit-box-flex:2;
            }
        .wrap div:nth-child(3){
            background: blue;
            -webkit-box-flex:1;
            }
    </style>
    <body>
        <div class="wrap">
            <div>第一</div>
            <div>第二</div>
            <div>第三</div>
        </div>
    </body>

box-flex：3,2,1 表示的占父级的几份，把父级分成几份，第一个占3份，以此类推

当设置了固定宽度的时候，后面的使用弹性布局，就会按照设置的比例分配

设置了margin之后，margin也是算在里面的
box-orient

  • 含义：定义子容器的排列方式

    • box-orient ：horizontal（子容器水平排列）

    • box-orient ：vertical（子容器垂直排列）

      • 不设置的时候默认为水平排列
box-direction

  • 含义：定义子容器的排列方式

    • box-direction：normal（正常的，按照从左到右或从上到下
      ）

    • box-direction：reverse（反转，同上相反）

box-align

  • 含义：子容器的垂直对齐方式

    • box-align：start(默认，居顶对齐)

    • box-align：center(居中对齐)

    • box-align：end(居底对齐)

box-pack

  • 含义：子容器的水平对齐方式

    • box-pack：start(默认，居左对齐)

    • box-pack：center(居中对齐)

    • box-pack：end(居右对齐)

    • box-pack：justify(水平等分父容器的宽度)
