js阶段第三篇

函数

函数定义：
function 函数名(参数列表) {
// 函数体
return 返回值;
}
function: 函数定义的关键字
函数名: 用来标识函数的名字
参数列表: 由外部向函数传入的内容
函数体: 一些用于执行的语句
return 返回值: 从函数内部向外传递内容
函数调用格式： 返回值=函数名（参数列表）
匿名函数：没有实际名字的函数，例如：
div.onclick=function(){
    //匿名函数
      //页面元素点击时，自动调用    
}

var fn=function(){
     //匿名函数
 }
  fn()//匿名函数调用

  • 1.无参数无返回值

    function fn1(){         
    //语句
    }

    • 2.有参数无返回值

      function fn2(argu){         
   //语句
   }

    • 3.无参数有返回值

      function fn3(){         
  return 123;//返回值
  }

    • 4.有参数有返回值

      function fn4(){         
     return 456;//返回值
     }

作用域

在行数内部声明的变量(局部变量)，在函数外部并不能访问
在函数外部声明的变量(全局变量)，在函数内部可以访问

math对象

  • 取最小的数

    var a=Math.min(1,2,3,4,5)
console.log(a);//结果为:1

  • 取最大的数

    var b=Math.max(1,2,3,4,5)
console.log(b);//结果为:5

  • 向上取整

    var c=Math.ceil(4.1)
console.log(c);//结果为:4
var d=Math.ceil(-4.1)
console.log(d);//结果为:-4

  • 向下取整

    var e=Math.floor(4.1)
console.log(e);//结果为:4
var f=Math.floor(-4.1)
console.log(f);//结果为:-5

  • 四舍五入

    var g=Math.round(4.1)
console.log(g);//结果为:4

  • 随机数，范围是0-1；不包括0 ，也不包括1

    var h=Math.random()
console.log(h);
console.log(h*(10+1));//0-10之间的随机数
console.log(h*(10+1)+10);//10-20之间的随机数
取一个很大范围内的随机数，函数的表达式如下：
第一种方法
function rand(min,max){
        var r;
        r=parseInt(Math.random()*(max-min+1)+(min));
        return r;
    }
    alert (rand(80,1200));
第二种方法
function rand(min,max){
        return parseInt(Math.random()*(max-min+1)+        (min));
    }
    var r=rand(80,1200);
    alert(r);
