文件打开模式

读取文件的四种方法(前提是有一个index.txt文件)

######fopen打开文件 fclose关闭文件

  • 第一种fread):读取整个文件(

    $fh=fopen("index.txt","a+");
$str=fread($fh,filesize("index.txt"));
fclose($fh);

  • 第二种(fgets):通过文件指针获取的是一行一行(则需要用一个循环实现全文的读取)，feof文件指针，如果文件指针指向结尾就是true

    $fh=fopen("index.txt","a+");
while(!feof($fh)){
    $str=fgets($fh);
        echo $str."<hr>";
}
fclose($fh);

  • 第三种:根据一行得到一个一个的数组，不需要配合feof与fopen，fclose

    $arr=file("index.txt");
var_dump($arr);

  • 第四种:常用的读取文件的方法 ，不需要feof与fopen，fclose

$str=file_get_contents("index.txt");
echo $str;

写入文件

  • w是替换 a是追加

  • 第一种:fwrite（需要fopen与fclose，可追加可替换）

$fh=fopen("index.txt","a");
fwrite($fh,"nihaoshijie");
fclose($fh);
$str=file_get_contents("index.txt");
echo $str;

  • 第二种:不需要fopen与fclose，直接替换

    file_put_contents("index.txt","1111");

复制、重命名、删除文件

<?php 
 header("Content-type: text/html; charset=utf-8");//解析中文
     //复制文件
     if(copy("index.txt", "test.txt")){
         echo "复制成功";
     }else{
         echo "复制失败";
     }

     //修改文件名
     if(rename("test.txt","copy.txt")){
         echo "修改成功";
     }else{
         echo "修改失败";
     }

     //删除文件
      if(unlink("copy.txt")){
         echo "删除成功";
     }else{
         echo "删除失败";
     }
?>

读取目录

  • 1.opendir()
    resource opendir ( string path [, resource context] )
    打开目录句柄;

  • 2.closedir()
    void closedir ( resource dir_handle )
    关闭目录句柄

  • 3.readdir() 一次只能获取一个

  • 4.scandir返回一个包含所有的目录和文件的数组;

例：

$dh=opendir(".");
while($files=readdir($dh)){
    echo $files."<hr>";
}
closedir($dh);
$arr=scandir("..");
print_r($arr);

  • 5.增加一个目录

    mkdir("haha");

  • 6.删除一个目录

    rmdir("haha");

其他文件操作函数

  • 1.filectime()获取时间戳

    date_default_timezone_set('PRC');//设置时区
$t=filectime("index.txt");
echo $t."<hr>";      
echo date("Y-m-d H:i:s",$t);

  • 2.fileatime() 返回文件的最后改变时间;

  • 3.filemtime() 返回文件的最后修改时间;

  • 注：”最后改变时间”不同于 “最后修改时间”。最后改变时间指的是对文件inode数据的任何改变，包括改变权限，所属组，拥有者等; 而最后修改时间指的是对文件内容的修改

  • 4.file_exists() 检查文件或目录是否存在，如果存在返回true, 否则返回false;

  • 5.is_readable() 判断文件是否可读，如果文件存在并且可读，则返回true;

  • 6.is_writable() 判断文件是否可写，如果文件存在并且可写，则返回true;

解析目录路径函数

  • 1.basename(),返回路径中的文件名部份，当指定了可选参数suffix会将这部分内容去掉;

  • 2.dirname(),返回路径中的目录部份;

  • 3.pathinfo(),返回一个关联数组，其中包括路径中的四个部份：目录名 dirname，文件名 filename， extension扩展名，basename文件名+扩展名(如果有扩展名的话)

    $path="/home/www/data/users.txt";
$filename=basename($path,".txt");//参数是去掉右面.txt
echo $filename;
$dirname=dirname($path);//得到目录结构
echo $dirname;
$arr=pathinfo($path);
var_dump($arr);

  • 4.chmod(),修改文件目录权限

    chmod("ccc",0755);//八进制数，正确的 mode 值 

[注意:]mode 参数包含三个八进制数按顺序分别指定了所有者、所有者所在的组以及所有人的访问限制。每一部分都可以通过加入所需的权限来计算出所要的权限。数字1表示使文件可执行，数字2表示使文件可写，数字4表示使文件可读。3=1+2(也就是可执行可写)，以此类推，最大可以到7
