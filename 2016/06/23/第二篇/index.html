css3的单位与响应式布局

倍数单位

1. em是相对单位

  • 相对父级设置和自己的字号设置

注意：h1~h6都有默认的1.5em,当不给h标签设置font-size，就会按照父级的1.5em计算，
当给h2设置了font-size,就会按照设置的倍数计算

<pre>
<style>
    .nav{
        width:500px;
        height:300px;
        border:1px red solid;
        font-size:12px;
    }
    .nav p{
        font-size:4em;
    }
    .nav h2{
        font-size:2em;
    }
</style>
<div class=“nav”>
    <p>世界你好</p>
    <h2>一切皆有可能</h2>
</div>
</pre>

上面的这样计算下来p标签的字号大小为：124=48px;h2标签的字号大小为122=24px，但是如果不给h2设置fong-size就会按照父级的1.5em设置，也就是12*1.5=18px;

2.rem是根单位

  • 相对于根字体(ps:根据HTML的字号）的大小来设置自己的字号大小

注意：系统默认的为16px(当HTML没有设置字体的大小就会按照默认的16px计算)，系统最小的字号为12px(ps:当在HTML中设置的字体小于12px,系统会按照最小的12px计算)

百分比单位

  • vw,vh是相对于窗口的单位(指宽度跟高度)

  • maxwidth maxheight minwidth minheight 这些是设置的最大的跟最小的宽高度（ps:不是百分比单位，是设置具体的像素）

响应式布局

  • 响应式布局是根据浏览器的界面大小来改变页面的大小，使用百分比单位，不设置固定的宽高，例：
 <pre>
   <style>
        body{
        margin:0;
        }
    ul{
        margin:0;
        padding:0;
        list-style-type: none;
        }
    .div1{
        width: 75%;
        max-width: 1200px;
        min-width: 600px;
        margin:100px auto;
        background: pink;
        overflow: hidden;
        }
    .div1 li{
        width: 31%;
        float: left;
        min-width: 300px;
        margin-right: 2%;

        }
    .div1 img{
        width:100%; 
        }
    </style>
   <body>
<ul  class="div1">
    <li>
        <img src="fengjing.jpg" alt="风景">
    </li>
    <li>
        <img src="fengjing.jpg" alt="风景">
    </li>
    <li>
        <img src="fengjing.jpg" alt="风景">
    </li>
    <li>
        <img src="fengjing.jpg" alt="风景">
    </li>
    <li>
        <img src="fengjing.jpg" alt="风景">
    </li>
    <li>
        <img src="fengjing.jpg" alt="风景">
    </li>
</ul>
</body>
    </pre>

运行之后的界面为三列缩小窗口会变成两列甚至是一列，拉大窗口会一直窗口的90%(在不超过设置的最大的宽度下)
