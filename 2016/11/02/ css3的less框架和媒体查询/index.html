css3的less框架和媒体查询

less文件

参考网址：http://www.bootcss.com/p/lesscss/

变量能够直接当做属性值使用
@color：green；
#header{
color：@color;
}
变量也能够在复合属性中使用
@color：green；
#header{
border:10px @color solid;
}
变量无参数的混用
@color:green;
.classA{
    background:@color;
}
#header{
    .classA;
}
变量有参数的混用
.wrap{
    .trs(all;3s;ease;0s);
}
.trs(@property;@duration;@timing-function;@delay){
    transition:@property @duration @timing-function     @delay;
}
嵌套规则
#header {
h1 {
    font-size: 26px;
    font-weight: bold;
}
p { font-size: 12px;
    a { text-decoration: none;
   &:hover { border-width: 1px }
  }
}
 }

函数和运算

我们可以做属性值和颜色的运算，这样就可以实现属性值之间的复杂关系。

@the-border: 1px;
@base-color: #111;
@red:  #842210;
#header {
color: @base-color * 3;
border-left: @the-border;
border-right: @the-border * 2;
}

媒体查询

方式一：在link标签里面写
<link rel="stylesheet" media="only screen and (min-    width:750px)" href="css/        c.css" />
方式二：在css里面写或者在style样式里面写
@media only screen and (min-width: ) {}
